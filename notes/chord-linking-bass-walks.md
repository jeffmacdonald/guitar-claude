# Chord Linking with Walking Bass Lines

Bass walks connect chords by stepping through notes on strings 5 or 6. Strum the chord on beat 1, pluck the walking bass notes individually on the remaining beats, then land on the next chord on beat 1 of the next bar.

## C → Am (walk down string 5: C → B → Bb → A)

```
e|--0-------------------0--|
B|--1-------------------1--|
G|--0-------------------2--|
D|--2-------------------2--|
A|--3---2---1---0---0------|
E|-------------------------|
    C               →   Am
```

## G → Em (walk down string 6: G → F# → F → E)

```
e|--3-------------------0--|
B|--0-------------------0--|
G|--0-------------------0--|
D|--0-------------------2--|
A|--2-------------------2--|
E|--3---2---1---0---0------|
    G               →   Em
```

## D → A (walk down string 5: D → C# → B → A)

```
e|--2-----------0--|
B|--3-----------2--|
G|--2-----------2--|
D|--0-----------2--|
A|------4---2---0--|
E|-----------------|
    D       →    A
```

## Tips

- Walking notes are single bass notes -- don't strum the full chord on those, just pluck the one string
- Once these feel natural, try reversing them (e.g. Am → C walking *up*) -- same notes, opposite direction
- These are from Justin Guitar Module 15 chord linking exercises (BG-1509)
